# CHIP-8 遊戲虛擬機器與 OS 啟動器 (C & SDL2)

**這個專案完全由ai制定主題，並實作**
本專案是一個使用 C 語言與 SDL2 媒體程式庫開發的 **CHIP-8 虛擬機器模擬器 (Emulator)**，並搭載了一個復古圖形化的 **OS Dashboard 啟動器**，以及一個運行在虛擬機器上的**互動式加減法計算機程式**。

本專案非常適合用於**系統程式 (System Programming)** 課程的專案實作，完整印證了底層系統的運作機制。

---

## 🎯 涵蓋之系統程式概念

* **CPU 指令週期 (Fetch-Decode-Execute Cycle)**：實作 CPU 從虛擬記憶體擷取 16-bit Opcode、進行解碼，並執行對應 35 個 CHIP-8 指令的循環。
* **虛擬定址與記憶體配置**：模擬 4KB 的 RAM 空間，包含字型檔（Font Set）載入至 `0x050`、ROM 二進位檔載入至 `0x200` 區段。
* **精準按鍵釋放偵測 (Key Release Spec)**：遵循標準規格，實作暫停 CPU 並等待按鍵「按下後放開」的阻斷式輸入偵測，徹底解決鍵盤重複讀取（Key Bounce）問題。
* **繪圖顯存與 XOR 碰撞偵測**：實作 `DXYN` 繪圖指令，透過對 64x32 顯存緩衝區進行位元 XOR 運算，並在像素重疊時自動設置 `VF` 暫存器（作為碰撞偵測，是開發遊戲的核心）。

## 📁 專案檔案結構

* [src/chip8.h](src/chip8.h)：定義 CPU 結構體（包括 V0-VF 暫存器、I 指標、PC、SP、堆疊、顯示緩衝區、鍵盤狀態與按鍵釋放追蹤）。
* [src/chip8.c](src/chip8.c)：實作虛擬硬體初始化、字型載入、ROM 載入與 35 個 Opcode 的邏輯解碼。
* [src/main.c](src/main.c)：實作 SDL2 視窗、動態渲染、60 FPS 幀率控制、以及帶有滑鼠選單的 OS Dashboard。
* [roms/ibm.ch8](roms/ibm.ch8)：134 位元組經典 IBM Logo 測試 ROM（用於驗證 CPU 繪圖與定址）。
* [roms/calc.ch8](roms/calc.ch8)：143 位元組自製加減法計算機 ROM（運行在 CHIP-8 虛擬機器上的組合語言程式）。
* [CMakeLists.txt](CMakeLists.txt) 與 [Makefile](Makefile)：提供跨平台的編譯組態。

## 🛠️ 編譯與安裝說明

### 1. 於 Linux / WSL 環境下 (推薦)
在執行編譯前，需要先安裝 SDL2 開發庫：

```bash
# 更新套件清單並安裝 SDL2
sudo apt-get update
sudo apt-get install -y libsdl2-dev

# 進入專案目錄並編譯
cd chip8_emulator
make
```

### 2. 於 Windows 環境下 (搭配 MSYS2 / MinGW64)
如果您在 Windows 下使用 MinGW 編譯，請先安裝 SDL2 套件：

```bash
# 在 MSYS2 終端機中執行
pacman -S mingw-w64-x86_64-SDL2
```
編譯指令可直接執行 `make`，或透過 CMake 進行建置。

## 🚀 執行與操作指引

編譯成功後會輸出可執行檔 `chip8_emu`，請在終端機輸入：

```bash
./chip8_emu
```

### 1. OS Launcher Dashboard (主選單)
程式啟動後會呈現圖形選單介面：
* **滑鼠點選**：將游標移至圖示上會觸發高亮邊框，**滑鼠左鍵點擊**即可載入該程式。
* **鍵盤快捷鍵**：在主選單直接按下數字鍵 **`1`**（啟動 IBM Logo 測試）或 **`2`**（啟動計算機）。
* **返回選單**：程式執行中按下鍵盤 **`ESC` 鍵**，即可立刻返回主選單。
* **關閉程式**：在主選單下按下 **`ESC` 鍵** 會關閉模擬器。

---

### 2. 鍵盤按鍵映射 (鍵盤控制)
模擬器已將 CHIP-8 的 16 鍵鍵盤映射至 QWERTY 鍵盤的左側：

```text
QWERTY 鍵盤對應        CHIP-8 虛擬按鍵
1   2   3   4    --->    1   2   3   C
Q   W   E   R    --->    4   5   6   D
A   S   D   F    --->    7   8   9   E
Z   X   C   V    --->    A   0   B   F
```

---

### 3. 🧮 互動式計算機操作步驟 (載入 Calculator 後)
1. **輸入第一個數字**：按下您鍵盤的數字鍵（`1` 至 `9`，按 **`X` 鍵** 代表 `0`），畫面左方會印出數字。
2. **輸入運算子**：
   * 按下 **`Z` 鍵** (CHIP-8 的 `A`)：代表 **`+` 加法**。
   * 按下 **`C` 鍵** (CHIP-8 的 `B`)：代表 **`-` 減法**。
3. **輸入第二個數字**：按下鍵盤數字鍵（例如 `3`）。
4. **輸出結果**：畫面會印出 `=` 並顯示十進位運算結果（例如 `5 + 3 = 08` 或 `9 - 2 = 07`）。
5. **清除重設**：按下鍵盤上的**任意按鍵**即可清除畫面，重新開始下一次計算。
