習題 2 -- 請重新設計一個全新的程式語言，並寫出其編譯器或解譯器 (建議用 AI)
要求：
編譯器：編譯後的目的碼，必須要能執行 (用虛擬機或直接執行）
解譯器：直接執行
指引：
先構想要設計的語言目標與功能
撰寫該語言的程式範例
設計 EBNF / BNF 語法
強型態/弱型態，編譯/解譯，目標檔（堆疊機/暫存器機/中間碼/LLVM IR），有沒有垃圾蒐集
讓 AI 用指定的語言(Python/JS/C/C++/Rust/Go/C#....) 寫專案 (可用網頁版 AI 或 IDE 版）
